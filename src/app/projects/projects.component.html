Raspberry Pi Based Home Automation System
The Project builds a Home Security System using a Raspberry Pi Model B+. It accepts a 4 digit password from the person entering the house. If the password is correct, the door is opened and the person is allowed in. If the password is entered wrong thrice, the image of the intruder is captured and is sent to the House Owner’s Phone through a private network. The House owner can check the image and choose to allow the person in or not. If allowed, the door would open.
Skills used: Python, Raspberry Pi, NoIR Cam, Flask Server.
Duration: January – May 2015
